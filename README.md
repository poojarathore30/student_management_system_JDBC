# Student Management System :  JDBC & MVC pattern

- Eclipse
- Tomcat Server 9.0
- Java 11 LTS version
- MySQL 5.0
- HTML 5.0 & CSS 3.0 & JS 
- Servelet, JSP, JDBC
- J2EE MVC
